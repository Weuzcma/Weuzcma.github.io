# 🚀 Space Explorer Website

## Overview
Space Explorer is an educational website that teaches users about Earth, Mars, Saturn, and the importance of space exploration. The website includes interactive animations, smooth scrolling, and a timeline of major space missions.

## Features
- 🌍 Earth information
- 🔴 Mars information
- 🪐 Saturn information
- 🚀 Space Exploration Timeline
- ⭐ Animated stars and smooth scrolling
- 📱 Responsive design

## Built With
- HTML5
- CSS3
- JavaScript
- Liveweave
- W3Schools Spaces
- GitHub Pages
- ChatGPT
- GitHub Copilot
